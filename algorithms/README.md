# Learning Order

```
1. q_learning        — Tabular RL on CliffWalking (discrete state + action)
2. dqn              — Neural net Q-values on highway-v0 (continuous state, discrete action)
3. policy_gradient  — REINFORCE on highway-v0 (continuous state, discrete action)
4. ppo             — Gaussian policy on racetrack-v0 (continuous lateral steering)
5. sac             — Maximum entropy on racetrack-v0 (continuous lateral steering)
```

# 学习顺序

```
1. q_learning        — 表格型 RL，CliffWalking 环境（离散状态 + 离散动作）
2. dqn              — 神经网络 Q 值，highway-v0（连续状态，离散动作）
3. policy_gradient  — REINFORCE 策略梯度，highway-v0（连续状态，离散动作）
4. ppo             — 高斯策略，racetrack-v0（连续横向转向控制）
5. sac             — 最大熵框架，racetrack-v0（连续横向转向控制）
```
